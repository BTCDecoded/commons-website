Why Bitcoin Commons

Modular Architecture

Decentralized governance through composable modules, enabling competitive discovery and user sovereignty without consensus layer changes.

Economic Sustainability

Self-funding through merge mining revenue, creating economic alignment between miners and development priorities without external dependencies.

Proven Governance Principles

Based on Elinor Ostrom's Nobel Prize-winning research on commons governance, enforced through Bitcoin's cryptographic primitives.

Bitcoin Commons moves governance from centralized control (few maintainers, single point of failure) to decentralized, forkable governance (distributed maintainers, multiple implementations, user choice). This architectural shift makes capture structurally difficult.

How to Adopt

Fork the Bitcoin Commons governance framework for your Bitcoin implementation. Multiple implementations can coexist using this model.

Step 1: Export Governance Ruleset

Use the ruleset export template to understand the governance configuration structure.

Step 2: Fork Governance Configuration

Fork the governance repository and customize for your organization (maintainers, thresholds, etc.).

Step 3: Set Up Cryptographic Enforcement

Implement cryptographic enforcement using the bllvm-sdk and governance-app as reference.

Step 4: Configure Governance Fork Mechanism

Set up the governance fork mechanism for user sovereignty. See GOVERNANCE_FORK.md for technical details.

Step 5: Join Implementations Directory

Add your implementation to the public registry via self-service pull request:

  1. Fork the governance repository
  2. Edit implementations-registry.yml and add your implementation entry
  3. Submit a pull request (GitHub will auto-populate the PR template)
  4. Automated validation checks YAML syntax, required fields, URL formats, and prevents duplicates
  5. If validation passes, PR merges automaticallyâ€”no approval needed

See IMPLEMENTATIONS_REGISTRY.md for complete instructions. This permissionless process demonstrates the framework's decentralized, cypherpunk-aligned nature.
